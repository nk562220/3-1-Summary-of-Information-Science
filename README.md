# 정보과학 학습 아카이브 (3학년 1학기)

3학년 1학기 정보과학 수업의 주차별 알고리즘 학습 내용과 직접 작성한 소스 코드, 그리고 최신기술·정보과학 수행평가를 정리한 웹 포트폴리오입니다.

작성: **3514 최은수**

## 주요 내용

- **주차별 알고리즘 정리** — 우선순위 큐와 힙, 해시 테이블, 문자열 탐색, 알고리즘 설계와 성능 분석, 탐욕 알고리즘, 분할 정복, 동적 계획법, 백트래킹
- **직접 작성한 코드 + 실행 결과** — 각 과제의 소스 코드와 콘솔 출력, SVG 시각화
- **인터랙티브 HTML/JS 시각화** — 미로 백트래킹, 그리드 선택 백트래킹, MapReduce Top-K, Behavior Tree 시뮬레이터를 브라우저에서 직접 조작 가능
- **최신기술과 정보과학 수행평가** — MapReduce(Divide & Conquer + Heap), Behavior Tree(Tree + DFS) 분석 및 구현

## 구성

| 파일 | 설명 |
| --- | --- |
| `index.html` | 웹앱 진입점 |
| `style.css` | 스타일 (글래스모피즘 라이트 테마) |
| `app.js` | 데이터 로딩·렌더링·인터랙션 로직 |
| `data.json` | 주차별 학습 내용 데이터 |
| `assets/` | 다이어그램 이미지 및 시각화 HTML |
| `*.ipynb`, `*.pdf` | 원본 노트북 및 수업 자료 |

## 로컬에서 실행

`fetch("data.json")`를 사용하므로 로컬 파일을 그냥 열면 안 되고, 간단한 웹 서버로 띄워야 합니다.

```bash
# Python이 설치돼 있다면
python -m http.server 8000
# 이후 브라우저에서 http://localhost:8000 접속
```
